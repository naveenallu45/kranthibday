# Birthday Wishes for Kranthi

A cinematic, multi-page birthday surprise website built with love by Sarayu.

## Quick Start

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173)

## Secret Codes

Any one of these unlocks the experience:

- `KRANTHI629`
- `MOMLOVE`
- `6292026`

## Experience Flow

One continuous journey — no navigation menu. Tap **Continue** to move through each moment:

1. Secret gate (password unlock)
2. Birthday welcome
3. Memories one by one (4 chapters)
4. Handwritten letter
5. Final surprise celebration

A thin progress bar at the top shows how far along the journey you are.

## Customization

- Replace placeholder images in `src/utils/constants.js`
- Edit the letter content in the same file
- Swap background music by updating the audio `src` in `src/components/MusicToggle.jsx`

## Build

```bash
npm run build
npm run preview
```
